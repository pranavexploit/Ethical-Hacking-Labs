# Ethical Hacking Labs

This repository documents hands-on work in penetration testing, including labs, notes, and machine writeups. The focus is on practical application of techniques across the full attack lifecycle in controlled environments.

---

## Repository Structure

- My-CTF-Boxes/
  Capture The Flag machine documentations

- Notes-HACKTHEBOX/
  Course-based material

- Practical-Labs/
  Focused cybersecurity labs and exercises

- Vulnerable-Machine-Writeups/
  Walkthroughs of vulnerable systems and CTF boxes

---

## Technical Scope

- Reconnaissance and information gathering
- Network scanning and host discovery
- Service enumeration and analysis
- Vulnerability identification
- Exploitation and initial access
- Post-exploitation and persistence
- Traffic interception and analysis
- Password cracking

---

## Tools and Environment

- Nmap
- Metasploit Framework
- Netcat
- Wireshark
- Kali Linux
- Bash scripting
- Python

## Purpose

The objective of this repository is to build and document a practical foundation in penetration testing. All work is conducted in authorized lab environments.
